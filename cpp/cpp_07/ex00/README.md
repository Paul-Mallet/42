# ex00 - Function Templates

This module is a remainder concerning **function overloading** and an introduction to templates dynamic typing. When to use **template** keyword and also what is great to be aware about template **arguments**.

## How to run?

```bash
	# using GNU Makefile builder
	make
	# doxygen documentation (html)
	make doc
	# catch2 test tools (v.1.12.0)
	make test
	# gdb debugger
	make debug
	# remove .o files
	make clean
	# remove doc/ and exec file
	make fclean
	# recompile the project
	make re
```
